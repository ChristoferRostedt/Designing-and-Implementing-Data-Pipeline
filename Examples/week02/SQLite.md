# Install SQLite
## Windows / masOS / Linux

- https://w.sqlite.org/download.html 
- Add the folder to your OS PATH environment variable so the salite3 command works in a terminal. You can chek where you installed the sqlite, with command: where sqlite3 
- Check installation with command: qlite3 --version 
- Install VS Code Extensions for SQLite 
- Create a new database (and later get into the database) in the terminal: sqlite3 firstExample.db 
- type .quit to get out
- type tables to check what tables have been created
